This is a simple Java console-based banking system created as a class project. The program allows users to log in using a valid account number and PIN, then perform banking transactions such as checking balances, making deposits, and withdrawing money. The application was built using multiple classes (BankAccount, Bank, and Main) to demonstrate object-oriented programming concepts in Java.
Three sample accounts are included for testing purposes:

This project demonstrates Java fundamentals including classes, methods, loops, conditionals, encapsulation and user input validation.
